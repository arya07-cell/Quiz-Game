# Java Quiz Application

## Overview

This is a simple **console-based Quiz Application** developed in Java. It presents multiple-choice questions to the user, accepts their answers, and provides immediate feedback. At the end of the quiz, the application displays the userâ€™s total score and a performance message.

## Features

* Multiple-choice questions with 4 options each
* Real-time answer validation
* Score calculation and final result display
* Easy to extend with more questions
* Clean and beginner-friendly Java code

## How to Run

1. Clone or download the repository
2. Compile the code using `javac QuizApp.java`
3. Run the program using `java QuizApp`
4. Answer the questions as they appear on the console

## Technologies Used

* Java (Core)
* Object-Oriented Programming
* Arrays and Loops
* User Input handling with Scanner

## Ideal For

* Java beginners looking to practice core programming concepts
* Students learning OOP and console application development
